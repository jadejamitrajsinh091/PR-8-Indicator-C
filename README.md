## 🧊 Q2 – Cube of 2D Array Using Pointer

<a href="https://onlinegdb.com/9wYcJLd98" target="_blank">Q-2</a>

### 🧠 Description:
This program takes a 2D array and calculates cube of each element using pointer.

### ⚙️ Logic:
- 2D array is passed as pointer
- Elements accessed using pointer arithmetic
- Each element is cubed

### ✅ Example:
```
Input:
2
1 2
3 4

Output:
1 8
27 64
```
